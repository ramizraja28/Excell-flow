How to Use Excel Flow
  1. Login — Enter username and password. Default is Rabbit / October28.
  2. Upload Files — Upload your Orbit export (source) and master Excel sheet (destination). Set which row your headers are on if needed.
  3. Map Fields — Auto-match columns are pre-filled. Use drag and drop or type coordinates manually. Search bar helps find columns fast.
  4. Row Options — Filter which rows to copy. Add new columns or label rows.
  5. Dry Run — Test the transfer without saving anything first.
  6. Transfer — Run the real transfer. Only adds data, never deletes anything.
  7. Download — Get the updated file, original backup, or a PDF report.
  8. Settings — Change username, password, theme, font, and shortcuts.
